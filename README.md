# 🎮 GAME-Z

**GAME-Z** is a browser-based mini gaming platform where users can play simple and fun games directly in their web browser without downloading anything.

🌐 **Live Website:** https://game-z-blush.vercel.app/

---

# 🚀 Features

* 🎮 Multiple browser-based mini games
* ⚡ Instant play – no downloads required
* 🖥️ Clean and responsive UI
* 📱 Works on desktop and mobile browsers
* 🎯 Simple navigation to access games quickly

Online game platforms allow users to enjoy games directly in their browser without installation, making gaming accessible and convenient.

---

# 🕹️ Available Games

The platform currently includes:

* ♟️ Chess
* 🐍 Snake Game
* 🎯 Other browser mini-games

More games can be added easily to expand the platform.

---

# 🛠️ Built With

* **HTML5**

* **JavaScript**
